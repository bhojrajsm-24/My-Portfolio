# 🌐 GitHub Pages Setup Guide

## Make Your Portfolio Live in 5 Minutes!

Your portfolio is on GitHub. Now make it accessible online with GitHub Pages.

---

## 📋 Quick Setup Steps

### 1. Go to Your Repository
```
https://github.com/bhojrajsm-24/My-Portfolio
```

### 2. Click "Settings"
- Top navigation bar
- Near the right side

### 3. Click "Pages"
- Left sidebar
- Under "Code and automation"

### 4. Configure Source
Under "Build and deployment":

**Source**: Deploy from a branch

**Branch**: 
- Select: `main`
- Folder: `/ (root)`
- Click **Save**

### 5. Wait
- Deployment takes 2-3 minutes
- Refresh the page

### 6. Your Portfolio is Live! 🎉
```
https://bhojrajsm-24.github.io/My-Portfolio/
```

## 🚀 Optional: Custom Domain

Want `yourname.com` instead of `github.io`?

### Steps:
1. Buy a domain (Namecheap, GoDaddy, etc.)
2. In GitHub Pages settings:
   - Click "Add a custom domain"
   - Enter your domain
   - Follow DNS configuration steps
3. Wait 24-48 hours for DNS propagation

---

## 🔄 Auto-Updates

Every time you push code to GitHub:
```bash
git add .
git commit -m "Update"
git push
```

GitHub Pages automatically rebuilds and updates your live site in 1-2 minutes!
