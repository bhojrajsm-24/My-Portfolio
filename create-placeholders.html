🎨 Portfolio Placeholder Images

Your portfolio is ready to use! However, you need to add your actual images to make it complete.

⚠️ Important: The portfolio currently uses placeholder image paths. You need to add real images for the site to display properly.

📋 Quick Setup Instructions:

  1. Take or find a professional photo of yourself
  2. Rename it to profile.jpg
  3. Place it in the assets/images/ folder
  4. Take screenshots of your 6 projects
  5. Name them project1.jpg through project6.jpg
  6. Place them in the assets/images/ folder
  7. Add your resume as assets/resume.pdf

📐 Recommended Image Sizes:

  • profile.jpg - 400x400 pixels (square)
  • project images - 800x600 pixels (landscape)

🎨 Tools to Create Placeholder Images:

  • Canva - Create custom images easily
  • Figma - Design professional mockups
  • Unsplash - Free high-quality stock photos
  • Placeholder.com - Generate simple placeholders
💡 Pro Tip: Use consistent styling across all project images for a more professional look. You can add borders, shadows, or mockup frames using tools like Figma or Canva.

✅ After Adding Images:

  1. Open index.html in your browser
  2. Check that all images load correctly
  3. Test on mobile devices
  4. Deploy to GitHub Pages, Netlify, or Vercel
