## 📋 Essential 5-Minute Setup

### 1. Add Images
Place these files in `assets/images/`:
- `profile.jpg` (your photo, 400x400px)
- `project1.jpg` through `project6.jpg` (screenshots, 800x600px)

### 2. Add Resume
- `assets/resume.pdf` (your CV)

### 3. Update Info (in index.html)
Find and replace:
- Your name
- Your email
- Your GitHub URL
- Your LinkedIn URL

### 4. Test
- Open `index.html` in browser
- Check all sections
- Test on mobile (F12 → Device Toolbar)

### 5. Deploy
Choose one:
- **GitHub Pages** (recommended - free, easy updates)
- **Netlify** (drag and drop)
- **Vercel** (one click)

See `DEPLOYMENT.md` for detailed steps.

## ⚡ Pro Tips for Success

### Do's ✅
- Use a professional photo
- Keep descriptions honest
- Update regularly (every 3 months)
- Use real project screenshots
- Proofread everything multiple times
- Test on multiple devices
- Get feedback from mentors

### Don'ts ❌
- Don't exaggerate skills
- Don't use stock photos for projects
- Don't leave lorem ipsum text
- Don't have broken links
- Don't ignore mobile testing
- Don't deploy without testing
- Don't forget to update resume
