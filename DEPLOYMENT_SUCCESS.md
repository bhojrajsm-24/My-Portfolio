## 🆘 Troubleshooting

### If GitHub Pages Doesn't Work:
1. Verify branch is `main`
2. Verify root folder is selected
3. Wait 5 minutes for deployment
4. Check repository is Public (not Private)
5. Clear browser cache

### If Images Don't Load:
1. Check file paths are correct
2. Verify images are in assets/images/
3. Check case sensitivity (GitHub is case-sensitive)

### If Resume Doesn't Download:
1. Verify PDF is in assets/resume/
2. Check file path in index.html
3. Clear browser cache
